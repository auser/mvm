mvm

A Rust CLI that manages Firecracker microVMs via Lima virtualization on macOS and Linux.

Install

One-liner (latest release)
curl -fsSL https://raw.githubusercontent.com/auser/mvm/main/install.sh | sh
Pin a version
MVM_VERSION=v0.1.0 curl -fsSL https://raw.githubusercontent.com/auser/mvm/main/install.sh | sh
Self-update
mvm upgrade

Quick Start

# From zero to SSH session in one command:
mvm dev

# Or step by step:
mvm bootstrap  # install Lima, create VM, download Firecracker
mvm start     # start microVM and SSH in
mvm ssh       # reconnect to running microVM
mvm stop      # shut it down

Commands

CommandDescription
mvm bootstrapFull setup from scratch — installs Lima via Homebrew, then runs all setup steps
mvm setupCreate Lima VM, install Firecracker, download kernel and rootfs
mvm devSmart entry point — detects state, auto-bootstraps if needed, launches microVM
mvm startStart the microVM and drop into interactive SSH
mvm stopStop the running microVM and clean up
mvm sshSSH into a running microVM
mvm statusShow status of Lima VM and microVM
mvm destroyTear down the Lima VM and all resources
mvm upgradeCheck for and install the latest version

How It Works

Three-Layer Stack

The CLI runs on your host, Lima provides a Linux VM with /dev/kvm, and Firecracker runs microVMs inside it.

Idempotent Setup

Every step checks whether it's already done before acting. Safe to re-run at any time.

Persistent VMs

Firecracker runs as a daemon. Exit SSH and the microVM keeps running. Reconnect with mvm ssh.

Self-Updating

Run mvm upgrade to pull the latest release from GitHub and replace the binary in-place.
